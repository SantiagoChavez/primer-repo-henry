📝 Comandos útiles de Git

📌 Inicialización y configuración git init Inicializa un repositorio nuevo en la carpeta actual. git clone Clona un repositorio remoto a tu máquina. git config --global user.name "Tu Nombre" git config --global user.email "tu@email.com" Configura tu identidad en Git. 📂 Estado y seguimiento de cambios git status Muestra el estado actual del repositorio. git add git add . Agrega archivos al área de staging. git restore Descarta cambios en un archivo (antes de hacer commit). 💾 Guardar cambios git commit -m "mensaje" Guarda los cambios con un mensaje. git commit -am "mensaje" Agrega y commitea archivos modificados (no nuevos). 🌿 Ramas (branches) git branch Lista las ramas. git branch Crea una nueva rama. git checkout Cambia de rama. git checkout -b Crea y cambia a una nueva rama. 🔀 Fusión de ramas git merge Fusiona una rama con la actual. 🌐 Trabajo con repositorios remotos git remote add origin Conecta tu repositorio local con uno remoto. git push origin Sube tus cambios al repositorio remoto. git pull Trae cambios del repositorio remoto y los fusiona. git fetch Trae cambios sin fusionarlos. 🕓 Historial git log Muestra el historial de commits. git log --oneline Historial resumido en una línea por commit. ⚠️ Deshacer cambios git reset --hard Elimina TODOS los cambios (usar con cuidado). git revert Deshace un commit creando uno nuevo. 🚀 Tips útiles Usá mensajes de commit claros y cortos. Hacé commits pequeños y frecuentes. Usá ramas para nuevas funcionalidades. Siempre hacé git pull antes de trabajar. 📚 Flujo básico de trabajo git add . git commit -m "mensaje" git push
