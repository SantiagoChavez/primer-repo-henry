# Comandos Útiles de Git

Este proyecto es una guía básica de comandos de Git organizada para consulta rápida. Surge como parte de una práctica para aprender control de versiones.

## Contenido

- Comandos de inicialización y configuración
- Seguimiento de cambios
- Manejo de commits
- Uso de ramas
- Trabajo con repositorios remotos
- Historial y deshacer cambios
- Flujo básico de trabajo

## Cómo usar

Abrí el archivo `index.html` en tu navegador para visualizar la guía de comandos.
